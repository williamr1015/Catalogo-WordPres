Este proyecto esta pensado en mostrar las diferentes formas en que podemos crear un catalogo en Wordpress
 y en el muestro tres formas en las que concidero se pueden realizar este tipo de catalogos. en la vista 
principal encontramos la forma mas comun que es utilizando las entradas y herraminetas simples que nos ofrece wordpres.
en la segunda llamada Catalogo muestro como se hace por medio de custom post y con la utilizacion de custom fields para algunos campos
y en la tercera ubicada en la vista Catalogo Productos muestro como hacer esta insercion de datos utilizando una base de datos la cual nos 
provee todos los datos y alli los renderizo.
por ultimo adjunto la base de datos utilizada en la cual se incluyen 4 tablas, una para poner los productos, otra para llevar control de los precios, 
otra lleva el control del inventario y la ultima se encarga de separar los productos por tipos.
